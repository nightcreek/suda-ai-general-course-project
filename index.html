项目说明

本项目面向大学新生，演示如何把学生画像、机器学习预测、学习计划、文件问答和 GeoGebra 数形结合讲题整合为一个智能学习伙伴原型。

建议演示顺序：学生画像 → 学习计划 → 文件问答 → 数形结合。

GitHub Pages 部署说明

本仓库首页为 index.html。上传到 GitHub 后，在 Settings → Pages 中选择 main / root 即可发布。
