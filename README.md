# 苏大人工智能通识选修课作业：智能学习伙伴原型

这是一个纯前端静态网页项目，适合通过 GitHub Pages 展示。

## 页面结构

- `index.html`：项目首页
- `student-profile.html`：学生画像与预测
- `learning-plan.html`：学习计划与复盘
- `ai-chat.html`：AI 对话与文件问答
- `math-geogebra.html`：AI 数形结合讲题
- `model-description.html`：模型说明与项目文档

## GitHub Pages 部署

在仓库 Settings → Pages 中选择：

- Source: Deploy from a branch
- Branch: main
- Folder: /root

然后访问生成的 Pages 链接。

## 说明

本项目是课程演示原型，不包含自建后端服务器。API Key 由用户自行输入，仅保存在当前页面运行时内存中。
